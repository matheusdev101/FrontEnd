O que é um Algoritmo?

Um algoritmo é uma sequência finita de passos organizados e bem definidos utilizados para resolver um problema ou executar uma tarefa.

Ele precisa ser claro, lógico e seguir uma ordem específica para que o resultado final seja correto.

No cotidiano, utilizamos algoritmos em diversas situações, como seguir uma receita, montar um móvel ou calcular o troco de uma compra.

Lógica de Programação

A lógica de programação é a capacidade de organizar pensamentos e instruções de forma estruturada para que o computador possa entender e executar.

Ela envolve raciocínio lógico, análise de problemas e organização de soluções passo a passo.

Sem lógica, não existe programação eficiente.

Estruturas Condicionais

As estruturas condicionais permitem que o algoritmo tome decisões com base em condições.

Elas verificam se algo é verdadeiro ou falso e executam ações diferentes dependendo do resultado.

IF, ELSE e ELSE IF

O comando IF executa um bloco de código caso uma condição seja verdadeira.

O ELSE executa outro bloco caso a condição seja falsa.

O ELSE IF permite testar múltiplas condições em sequência.

Exemplo lógico:

Se a nota for maior ou igual a 7 → aprovado.

Senão → reprovado.

Teste de Mesa

O teste de mesa é uma técnica usada para simular manualmente a execução de um algoritmo.

Nele, acompanhamos os valores das variáveis passo a passo para verificar se o resultado final está correto.

Essa técnica ajuda a identificar erros de lógica antes mesmo da execução no computador.
